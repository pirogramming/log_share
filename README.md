## 🖍 App 주요 기능
1. log_share, main
    * 프로젝트 setting, 메인화면
1. accounts
    * 로그인, 회원가입, 비밀번호 변경, 프로필 작성
1. group_management
    * 그룹 생성, 그룹 삭제
    * 그룹 관리자 기능: 그룹 정보 변경, 회원 강제 탈퇴, 가입 요청 수락/거절
    * 그룹 검색, 그룹 가입 요청<br>
1. myprofile
    * 스크랩(북마크)
    * 사용자별 페이지: 소개, 활동기록 타임라인, 활동기록 리스트
    * 프로필 수정, 상세보기 페이지<br>
1. post
    * 활동기록 포스트 작성, 수정, 삭제
    * 포스트 리스트를 보여줄 때 무한 스크롤 가
1. search
    * 같은 그룹 내의 사람들의 활동기록 검색 페이지
    * 포스트의 제목/내용, 태그, 작성자 중 선택해서 검색 가능
    * 그룹, 포스트의 카테고리, 활동기간 필터 적용 가능<br>
